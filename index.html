Arrays

Arrays are a way to group multiple items together. They're a way to represent a list of things.

Loops

Loops let us repeat a line of code multiple times. Like the if statement, a loop is one of the most useful constructs in programming.

While Loops

While loops will run until the condition in their brackets is false. They're useful when you don't know how many times you want to run the loop.
